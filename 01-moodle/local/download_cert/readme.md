Certificado mutual pdf:

Para descargar el certificado dentro del curso hacemos clic sobre el link del menú lateral con el nombre "Descargar certificado", el cual nos enviara a la dirección del pdf como por ejemplo:
https://site/local/download_cert/donwloadcertificate.php?courseid=2

El cual tiene los datos:  Nombre curso, Fecha Curso (Fecha de fiscalización), Fecha de caducidad, Nombre empresa, Calificación, Estado.

Para configurar la fecha de caducidad dentro del curso hacemos clic sobre la opción "Configurar caducidad de certificado" y configuramos los años de caducidad para este curso. 

Un ejemplo de la ruta para configurar es la siguiente:
https://site/local/download_cert/configyear_certificate.php?courseid=2

La configuración se hará en base a años y en caso de no estar configurada esta opción el valor base sera 3 años.
